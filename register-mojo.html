otka:register

Full name:

com.okta:okta-maven-plugin:0.3.1:register

Description:

Signs up for a free Okta Developer Account. Prompts for basic user info, accounts will be activated via email.

Attributes:

  • Executes as an aggregator plugin.

Optional Parameters

Name Type Since Description
<company> String - Company / organization used when registering a new Okta account.
User property is: company.
<demo> boolean - Set demo to true to force the prompt to collect user info. This makes for a better demo without needing to create a new Okta Organization each time.

NOTE: Most users will ignore this property.


Default value is: false.
User property is: okta.demo.
<email> String - Email used when registering a new Okta account.
User property is: email.
<firstName> String - First name used when registering a new Okta account.
User property is: firstName.
<lastName> String - Last name used when registering a new Okta account.
User property is: lastName.
