otka:login

Full name:

com.okta:okta-maven-plugin:0.3.1:login

Description:

Authenticates the Okta CLI and Okta Maven Plugin with an exiting Okta Organization. To sign up for a new or use okta:register.

Attributes:

  • Executes as an aggregator plugin.

Optional Parameters

Name Type Since Description
<apiToken> String - Okta API token.
User property is: apiToken.
<orgUrl> String - The base Okta Organization URL. Okta Developer Orgs are commonly in the format of: https://dev-12345.okta.com/.
User property is: orgUrl.
