okta:web-app

Full name:

com.okta:okta-maven-plugin:0.4.0:web-app

Description:

Creates a new Okta OIDC Web Application for use with your Web Application and writes an .okta.env file with it's configuration.

NOTE: You must have an existing Okta account to use this Mojo, use the goals: okta:register or okta:login.

To create other types of applications on the command line see the Okta CLI.

Attributes:

  • Executes as an aggregator plugin.

Optional Parameters

Name Type Since Description
<authorizationServerId> String - The id of the authorization server.
Default value is: default.
User property is: authorizationServerId.
<oidcAppName> String - The Name / Label of the new OIDC application that will be created. If an application with the same name already exists, that application will be used.
Default value is: ${project.name}.
User property is: oidcAppName.
<redirectUri> String - The redirect URI used for the OIDC application.
Default value is: http://localhost:8080/callback.
User property is: redirectUri.
