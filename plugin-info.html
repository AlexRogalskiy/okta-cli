Plugin Documentation

Goals available for this plugin:

Goal Description
otka:setup Creates a free Okta Developer account and registers / configures a new OIDC application. To signup for an account without using this plugin visit developer.okta.com.

If you have an existing Okta account, you can configure this plugin to use it by configuring a ~/.okta/okta.yaml configuration (or equivalent, see the Okta SDK configuration reference for more info).

TL;DR: If you have an exiting account create a ~/.okta/okta.yaml with your URL and API token:


okta:
  client:
    orgUrl: https://{yourOktaDomain}
    token: {yourApiToken}

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.5.0
JDK 1.8
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>com.okta</groupId>
          <artifactId>okta-maven-plugin</artifactId>
          <version>0.1.0-SNAPSHOT</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>com.okta</groupId>
        <artifactId>okta-maven-plugin</artifactId>
        <version>0.1.0-SNAPSHOT</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
